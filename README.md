# Strichmännchen Disco Runner Pro

Ein actiongeladenes Jump-and-Run-Spiel im Endless-Runner-Stil, das im Browser läuft. Weiche bewaffneten Strichmännchen-Gegnern aus, sammle Power-Ups und schalte neue Skins frei, während sich der Hintergrund im Disco-Stil dreht!

## Features

*   **Vollbild-Modus:** Das Spiel passt sich automatisch an die Größe Ihres Browserfensters an.
*   **Dynamische Schwierigkeit:** Das Spiel wird schneller, je länger Sie spielen.
*   **Freischaltungen:** 6 verschiedene Skins, die Sie durch das Erreichen von Highscores freischalten können.
*   **Soundeffekte:** Einfache Töne für Sprünge, Power-Ups und Game Over (abschaltbar).
*   **Barrierefreiheit:** Ein spezieller Farbblinden-Modus mit optimierter Farbpalette.
*   **"Bullet-Time" K.O.:** Ein dramatischer Zeitlupeneffekt, wenn Sie getroffen werden.
*   **Verschiedene Gegner:** Bodeneinheiten mit Waffen und fliegende Dronen.

## Wie man spielt

1.  Öffnen Sie die Datei `index.html` in Ihrem Webbrowser (Chrome, Firefox, Safari, Edge, etc.).
2.  Klicken Sie auf **"Spiel Starten"** oder drücken Sie die **Leertaste**.
3.  Drücken Sie die **Leertaste** oder **Pfeil nach oben**, um zu springen.
4.  Weichen Sie den roten Gegnern aus.
5.  Sammeln Sie grüne Power-Ups für einen Super-Sprung.

## Lokale Ausführung

Das Spiel benötigt keinen Webserver. Sie können es direkt in Ihrem Browser starten:

1.  Stellen Sie sicher, dass alle vier Dateien (`index.html`, `style.css`, `game.js`, `README.md`) im selben Ordner liegen.
2.  Doppelklicken Sie auf `index.html`, um das Spiel zu öffnen.

## Als App veröffentlichen

Da das Spiel ausschließlich auf HTML, CSS und JavaScript basiert, kann es mithilfe von Tools wie **Capacitor** oder **Cordova** relativ einfach in eine native mobile App (Android/iOS) verpackt und in den App Stores veröffentlicht werden.
